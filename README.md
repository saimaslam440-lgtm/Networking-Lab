# Computer Networking Lab

A comprehensive collection of Computer Networking lab materials, notes, Wireshark captures, protocol explanations, and filtering demonstrations. This repository is designed to help students understand networking concepts through theory and practical packet analysis.

---

##  Repository Contents

| File | Description |
|------|-------------|
| `Readme.md.txt` | Repository overview |
| `Commands.md.txt` | Common networking commands used during the lab |
| `OSI-Model.md.txt` | Detailed explanation of the OSI Model |
| `TCP-IP-model.md.txt` | TCP/IP Model overview and comparison |
| `Protocols.md.txt` | Networking protocols and their functions |
| `Wireshark.md.txt` | Introduction to Wireshark and packet analysis |
| `Lab-Structure.jpg` | Networking lab structure diagram |
| `Wireshark interface.jpg` | Wireshark interface overview |
